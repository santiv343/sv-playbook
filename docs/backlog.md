# Idea Backlog

Ideas are never killed — they are recorded here unvalidated. **An entry is
not a commitment** (PRINCIPLE-008): an idea graduates to a plan only when a
real project demonstrates the pain it solves, and the graduation note must
name that evidence. Review this file at every project closure/retro.

Status values: `unvalidated` · `scheduled-v2` (already in the spec's v2
scope) · `graduated` (moved to a plan; keep the row for traceability) ·
`obsolete` (superseded by a later decision; kept as a tombstone, never silently removed).
